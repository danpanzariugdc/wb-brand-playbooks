Use this reference whenever you're writing for WhichBingo. Every page has a primary keyword to target, secondary keywords to weave in naturally, and approved anchor text to use when linking.

Anchor Text Rules

How to link. Every time.

✅ Do This

→Use the primary keyword as your anchor when linking to a page for the first time.
→Use approved anchor text from the list on each card — these are pre-approved by the SEO team.
→Vary anchors across a piece — don't use the same anchor text twice for the same page.
→Make anchors read naturally in context. If it feels forced, rewrite the sentence around it.
→Planned URLs not yet live should still be referenced — your editor will activate the link.

🚫 Never Do This

→Never use click here, read more, or this page as anchor text.
→Never use anchor text that isn't on the approved list without checking first.
→Don't stuff multiple keywords into a single anchor. Keep it to 2–4 words max.
→Don't link the same anchor text to two different pages in one piece.

📐 How to Read Each Card

→Primary keyword — the main term this page targets. Lead with this.
→Secondary keywords — supporting terms to weave in naturally throughout your copy.
→Approved anchor text — the pre-approved list of anchor phrases to use when linking to this page.
